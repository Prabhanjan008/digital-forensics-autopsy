# Digital Forensic Investigation using Autopsy

## Overview
This project involves performing a digital forensic investigation on disk image files using Autopsy. The goal was to analyze system artefacts, recover deleted files, and examine user activity to identify potential evidence.

## Objectives
- Analyze disk image files
- Recover deleted data
- Examine web activity and USB usage
- Understand forensic investigation workflow

## Tools Used
- Autopsy v4.19
- Disk Image Analysis

## Key Learnings
- Digital forensic methodology
- Deleted file recovery
- Evidence documentation
- User activity analysis

## Type
Academic Digital Forensics Lab Project
